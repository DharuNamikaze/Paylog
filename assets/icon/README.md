# PayLog App Icon

This directory contains the app icon assets for PayLog.

## Current Status
- Using default Flutter launcher icon
- Custom icon can be added later using flutter_launcher_icons package

## To add custom icon:
1. Create a 1024x1024 PNG image named `app_icon.png`
2. Uncomment the flutter_launcher_icons configuration in pubspec.yaml
3. Run: `flutter pub get`
4. Run: `flutter pub run flutter_launcher_icons:main`

## Icon Design Guidelines
- Simple, recognizable design
- Works well at small sizes
- Represents financial/transaction tracking
- Green color scheme to match app theme
